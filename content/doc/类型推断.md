# 特殊类型

## 类型推断与类型别名
```ts
//正常来说ts会自动推断类型
let a = 1
let b = 'text'

type a = string | number //用于类型别名
type c = a & any
type teng = {
    name: 'tzh',
    age: 26,
    sex: 'male'
}
```

## never类型
- never类型作为底层类型(顶层any, unknown),一般表示不存在的情况,用来兜底
```ts
//  这里用void也行，但这个函数会报错，根本没有返回值，而不是返回值是空，所以void不合适
function test(): never{
    throw new Error()
}

// 假设随后功能扩充新加了篮球字段，但是某一个开发不知道原来的功能函数，这时候赋值的时候兜底逻辑因为是never类型，所以就会报错，提示开发
type A = '唱'|'跳'|'rap'|'篮球' 
function kun (value:A) {
    switch (value) {
        case "唱":
            break
        case "跳":
            break
        case "rap":
            break
        default:
            //兜底逻辑
            const error:never = value;
            break
    }
}
```

## Symbol
- Symol是ES6新增的特性，用来标识“唯一”，主要解决对象属性重复的问题
```ts
let a = Symbol('1')
let b = Symbol('1')
console.log(a === b) // false

//新方法Symbol.for会先去全局查找有没有对应key值的symbol，如果有直接返回该Symbol，没有则创建新的
Symbol.for('1') === Symbol.for('1') // true
```
