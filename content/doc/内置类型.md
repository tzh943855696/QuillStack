# 内置类型

## ECMA
```ts
let num:Number = new Number(1)
let date:Date = new Date()
let ref: RegExp = new RegExp(/\-w/)
let error = new Error('错了')
let xhr: XMLHttpRequest = new XMLHttpRequest()
```

### DOM
```ts
let div:HTMLDivElement  = document.querySelector('div')
let input:HTMLInputElement  = document.querySelector('input')

//普通的
let section: HTMLElement = document.querySelector('section')
let footer: HTMLElement = document.querySelector('footer')

//集合
let nodeList:NodeList = document.querySelectorAll('div')
let nodeListB: NodeListOf<HTMLDivElement | HTMLElement> = document.querySelectorAll('div footer')
```

### JS内置
```ts
let local:Storage = localStorage
let lo:Location = location
let promise:Promise<number> = new Promise((r) => r(1) )
```

# 元组(tuple)
```ts
let arr:[string,number] = ['1',1]
// 这时候仍然可以改值或者push
arr.push(1)
arr[0] = 1

// 这样就可以不允许改值，push
let arr: readonly [string,number] = ['1',1]

// 让item变为可选的
let arr: [x: string,y?: number] = ['1']
```
